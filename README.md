# Personal Portfolio

A clean, multi-page portfolio website built with plain HTML, CSS, and JavaScript — no frameworks required.

## Pages
- **Home** — intro photo + summary
- **About** — bio, timeline, interests
- **Resume** — experience, education, skills
- **Projects** — showcase of work
- **Blog** — notes and writing

## Features
- Slide-down navigation menu connecting all pages
- Fully responsive layout
- No build tools or dependencies — just open `index.html`
